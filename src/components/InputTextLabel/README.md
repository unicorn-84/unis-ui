# InputTextLabel

> An input label.

## API

### `label`

The label text.

**Type:** `string`

**Required:** `true`

#### Examples

```js
<InputTextLabel label="Username" />
```

### `size`

The size of the component.

**Type:** `sm | md`

**Default:** `md`

**Required:** `false`

#### Examples

```js
<InputTextLabel label="Username" size="sm" />
```
