# InputTextHint

> An assistive text for the user.

## API

### `text`

The assistive text.

**Type:** `string`

**Required:** `true`

#### Examples

```js
<InputTextHint text="An assistive text for the user" />
```

### `error`

If true, the component is displayed in an error state.

**Type:** `boolean`

**Required:** `false`

**Default:** `false`

#### Examples

```js
<InputTextHint text="Validation message text" error />
```
