# InputBase

> An base input.

## API

### `size`

The size of the component.

**Type:** `sm | md`

**Default:** `md`

**Required:** `false`

#### Examples

```js
<InputBase size="sm" />
```

### `error`

If true, the component is displayed in an error state.

**Type:** `boolean`

**Required:** `false`

**Default:** `false`

#### Examples

```js
<InputBase error />
```
