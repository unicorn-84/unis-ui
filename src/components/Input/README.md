# Input

> A text link.

## Usage

```js
import { Input } from '@web-uni/unis';

const App = () => {
  return <Input />;
};
```

## API

### `label`

The label text.

**Type:** `string | undefined`

**Default:** `undefined`

**Required:** `false`

#### Examples

```js
<Input label="Password" type="password" name="password" />
```
